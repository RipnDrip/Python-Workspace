Project GDD

# Event Action
Login Client is prompted to log in as a user or admin. If the user has an account, use Login ID and pass to access account. If it is a new user, use registration form to create a new account. Once registered, the user may login as normal.
Login/Registration Confirmation Send Confirmation when the user is registered or logged in successfully. Otherwise an error message is returned.
Registration Form User enters Name, password, age, weight, height, and credit card information. Uses user information to calculate prices based on Age, weight, and height. Stores prices for each individual into the database once calculated.
Search Page Contains available priced packages based on user profile. User may add/remove products to their own shopping cart.
Overview Page Contains summary of available packages as well as more detailed information about each package.
Shopping Cart Uses the user's added product database to calculate total price based on the chosen package(s). Once calculated the user can proceed to checkout screen and purchase the desired packages.
Checkout Displays a user cart based on saved information from the database. The user can confirm the purchase and use their specific credit card information to purchase what is in the cart.
